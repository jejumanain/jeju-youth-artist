# ✏️ **제주 청소년 작가 클럽** (Jeju Youth Artist Club)

**"서귀포를 넘어 제주 전역으로, 청소년들의 상상력이 작품이 되는 공간"**

제주 청소년 작가 클럽은 **제주만화인연대**가 운영하는 청소년 창작 공동체입니다. 기존 서귀포청소년만화동아리의 운영 경험을 바탕으로 제주 전 지역으로 범위를 확대하여, 온·오프라인 병행 멘토링과 지역 거점별 동아리 육성을 목표로 합니다.

## 🌟 **프로젝트 배경 및 목적**

- **지역적 확장**: 서귀포 지역 중심에서 제주 전 지역으로 활동 범위 확대

- **생태계 구축**: 현업 작가(멘토)와 청소년 작가(멘티)를 연결하여 제주 만화 생태계의 선순환 구조 확립

- **지속 가능성**: 작가 1인 중심의 운영 한계를 극복하고, 체계적인 운영 전략과 활동비 지원을 통해 지속 가능한 교육 환경 제공

## 📅 **운영 방식** (Hybrid Model)

- **온라인 모임**: 월 2회 (토요일 10:00 ~ 12:00) - 시공간 제약 없는 창작 공유

- **오프라인 워크샵**: 월 2회 (토요일 10:00 ~ 12:00) - 현장 멘토링 및 실습

- **순회 상담**: 4월 모집 기간 중 제주 전 지역 순회 상담 진행

- **정식 운영**: 매년 5월부터 본격적인 창작 활동 시작

## 🚀 **주요 특징**

- **전문 멘토링**: 제주만화인연대 소속 현업 작가들이 직접 지도

- **지역 거점화**: 일정 인원 이상 모집 시 지역 동아리로 분리 및 전담 지도 작가 배치

- **유료 멤버십**: 활동비(페이) 지급을 통한 교육 퀄리티 보장 및 책임감 있는 참여 유도

- **전시 및 출판**: 정기적인 전시 참여 및 크라우드 펀딩을 통한 작품집 발간

## 🛠 **기술 스택**

- **Frontend**: HTML5, Tailwind CSS, JavaScript (Vanilla JS)

- **Visualization**: Chart.js

- **Deployment**: GitHub Pages
